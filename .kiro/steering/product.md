# Product Overview

## 概要

「引きこもりエンジニアの徒然ログ」は、技術系記事を自由に記録・公開するための個人ブログアプリ。
Zenn などのプラットフォームのガイドラインに縛られず、自分のペースでアウトプットを蓄積することを目的とする。

## コアケイパビリティ

- **記事の閲覧**: Markdown ファイルを静的生成し、シンタックスハイライト付きで配信
- **目次自動生成**: h2・h3 見出しから TOC を自動生成。デスクトップはスティッキーサイドバー、モバイルは折りたたみ形式で表示
- **タグ・カテゴリ分類**: 記事にタグとカテゴリを付与し、絞り込み表示が可能
- **クライアントサイド検索**: ビルド時生成の `search-index.json` をフェッチして軽量検索を実現
- **閲覧数カウント**: Upstash Redis（HTTP REST）によるベストエフォート方式の閲覧数表示
- **ダークモード**: `next-themes` を使ったシステム連動（🖥）・ライト固定（☀️）・ダーク固定（🌙）の 3 状態切り替え対応

## ターゲット

- **主利用者**: 筆者本人（記事の管理・閲覧）
- **副次利用者**: ブログを読みに来る読者

## コンテンツ管理

記事は `content/posts/*.md` に Markdown ファイルとして管理する。
フロントマターに `published: false` を設定した下書きはビルド対象から除外される。
`git push` → GitHub → Vercel の自動デプロイにより記事が公開される。

## 価値提案

プラットフォームに依存せず、コードと同じ Git ワークフローで記事を管理できる。
静的生成により高速・低コストで運用でき、Lighthouse Score 90+ を維持する。
